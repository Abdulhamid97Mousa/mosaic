# BARLOG Worker Dependencies
# Install with: pip install -r requirements/balrog_worker.txt
#
# This file contains dependencies for the BARLOG (BALROG LLM) worker,
# which runs LLM-based agents on BALROG benchmark environments:
# - BabyAI (MiniGrid navigation)
# - MiniHack (NetHack procedural dungeons)
# - Crafter (open-world survival)
#
# See: https://github.com/balrog-ai/BALROG

# Include base requirements
-r base.txt

#
# 1. Install the worker:
#    pip install -e 3rd_party/balrog_worker

# 2. Set your API key in .env:
#    OPENAI_API_KEY=sk-...

#    ANTHROPIC_API_KEY=sk-ant-...

#    balrog-worker --run-id test123 --env babyai --task BabyAI-GoToRedBall-v0
